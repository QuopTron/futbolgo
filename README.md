# FutbolGO 🎯

Ver partidos y streams deportivos en vivo con AD-Blocker potente.

## Features

- 🎬 Reproductor de streams en vivo (HLS/MP4/WebView)
- 🚫 Ad-Blocker potente integrado
- 📺 Canales y eventos en vivo
- 🌍 Soporte multi-idioma (ES, EN, PT)
- 🔄 Sistema de fallback para streams no disponibles
- 📱 Diseño glass minimalista
- ✨ Actualizaciones automáticas de la app

## Installation

### Android

Descarga la APK desde [Releases](https://github.com/flox-app/futbolgo/releases/latest)

### Desktop

1. Instala Flutter: https://flutter.dev/docs/get-started/install
2. Clona el repo
3. `flutter pub get`
4. `flutter run`

## Build APK

```bash
flutter build apk --debug
flutter build apk --release
```
